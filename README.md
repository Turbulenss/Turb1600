# turb1600

`turb1600` is an **experimental sponge-based cryptographic hash
function** featuring a **1600-bit internal state** and a **1024-bit
output**.

The project is designed for **research, experimentation, and analysis**,
with an emphasis on clarity, determinism, and cross-language
reproducibility.

------------------------------------------------------------------------

## ⚠️ Project Status

> **Experimental / Research Only**

`turb1600` is under active development.\
The algorithm, parameters, and public interfaces **may change**.

**Do NOT use this implementation in production or security-critical
systems.**

------------------------------------------------------------------------

## Features

-   Sponge construction inspired by Keccak-style designs
-   1600-bit internal state (25 × 64-bit lanes)
-   1088-bit rate / 512-bit capacity
-   1024-bit output (128 bytes)
-   Deterministic, stateless permutation
-   Domain-separated initialization
-   Bit-for-bit compatible Rust and Python implementations
-   Reference test vectors (KATs)

------------------------------------------------------------------------

## Design Overview

`turb1600` follows a classic sponge structure:

-   **Absorb:** input message blocks are XORed into the state at the
    rate
-   **Permute:** a custom 64-bit-lane permutation provides diffusion and
    nonlinearity
-   **Squeeze:** output is extracted from the rate portion of the state

### Core Parameters

  Parameter      Value
  -------------- -----------------------
  State size     1600 bits
  Lane size      64 bits
  Rate           1088 bits (136 bytes)
  Capacity       512 bits
  Output size    1024 bits (128 bytes)
  Full rounds    32
  Final rounds   4

### Initialization

The state is initialized using a **domain-separated tag**:

    turb1600:v2|state=1600|rate=1088|capacity=512|out=1024

## Rust Usage

``` rust
use turb1600::turb1600_hash;

let digest = turb1600_hash(b"hello world");
```

------------------------------------------------------------------------

## CLI Usage

``` bash
cargo build --release
turb1600 "hello"
turb1600 --hex deadbeef
turb1600 --file message.bin
turb1600 --tag context "message"
```

------------------------------------------------------------------------

## Python Reference

``` python
from turb1600 import turb1600_hash
print(turb1600_hash(b"abc").hex())
```
